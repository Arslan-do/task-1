# Kelvin Kramer - Freelancer Portfolio

Верстка хедера и баннера для портфолио фрилансера по макету Figma.

## Что реализовано

✅ **Хедер:**

- Логотип "Kelvin." с желтой точкой
- Навигация: Home, Works, Services, About, Contact
- Адаптивное hamburger меню для мобильных устройств
- Фиксированная позиция при скролле

✅ **Баннер:**

- Двухколоночная раскладка на десктопе (текст слева, фото справа)
- Вертикальная раскладка на мобильных (фото сверху, текст снизу)
- Заголовок "Kelvin Kramer"
- Подзаголовок "Designer & Developer"
- Описательный текст

## Установка

1. Установите зависимости:

```bash
npm install
```

2. Запустите компиляцию SCSS (в режиме наблюдения):

```bash
npm run sass
```

3. Откройте `index.html` в браузере

## Структура проекта

- `index.html` - главная страница
- `scss/style.scss` - стили (SCSS)
- `css/style.css` - скомпилированные стили
- `js/script.js` - JavaScript для мобильного меню
- `package.json` - зависимости проекта

## Технологии

- HTML5
- SCSS/CSS3
- JavaScript (для мобильного меню)

## Загрузка на GitHub

1. Создайте новый репозиторий на GitHub
2. Выполните команды:

```bash
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
git branch -M main
git push -u origin main
```

## Добавление коллаборатора

После создания репозитория:

1. Перейдите в Settings → Collaborators
2. Нажмите "Add people"
3. Введите GitHub username коллаборатора
4. Выберите роль (обычно "Write" для возможности комментировать код)
